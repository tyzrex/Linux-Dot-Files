# i3-Greeny

### System Information

- OS : Pop OS
- WM : i3
- Music Player : Spotify ( theme with spicetify )
- Launcher : Rofi
- File Manager : Thunar
- GTK-theme : Greeny (Custom made)
- Terminal : kitty
